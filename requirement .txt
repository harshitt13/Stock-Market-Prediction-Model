Required Python Libraries:
PyTorch - For implementing Linear Regression and LSTM models.
Pandas - For data manipulation and analysis.
Numpy - For numerical operations.
Matplotlib - For plotting and visualizing stock prices.
Scikit-learn - For splitting the dataset and basic machine learning operations (such as Linear Regression).
Yfinance - For fetching historical stock data.
Joblib - For saving and loading Linear Regression models.
Tqdm - For displaying progress bars during model training
Typing - For  specifying the types of variables, function parameters, and return values, improving code readability and type-checking during development

You can run the command given below in terminal to install all necessary libraries.
pip install torch pandas numpy matplotlib scikit-learn yfinance joblib tqdm typing
